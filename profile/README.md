# VTC Woerden

Welcome to the GitHub organization of **VTC Woerden**, a local volleyball club based in Woerden, The Netherlands.

This organization contains repositories for wordpress plugins used on the websites [vtcwoerden.nl](https://vtcwoerden.nl) and [vtcbeach.nl](https://vtcbeach.nl).

## Contributing

Contributions are welcome, especially from club members and volunteers.

To get started:

- Clone one of the `wp-plugin-*` repositories
- Run a local WordPress environment (for example [Local WordPress](https://localwp.com/))
- Activate the plugin in your local WordPress instance
- Make your changes and test them locally
- Create a pull request
